# Reporteador de Información del Sistema en Python

## Autor
Maldonado Morales Francisco Eliud

## Descripción
Proyecto académico que reporta información del sistema usando Python
con componentes de bajo nivel en ARM64 Assembly.

## Tecnologías
- Python 3.10+
- ARM64 Assembly
- C (capa intermedia)

## Instrucciones de uso
1. Clona el repositorio
2. Ejecuta: `python src/main.py`
