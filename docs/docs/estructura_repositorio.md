# Estructura del Repositorio

## Árbol de carpetas
practica-codex-documentacion/
├── README.md
├── docs/
│   ├── propuesta.md
│   ├── caso_de_uso.md
│   ├── estructura_repositorio.md
│   └── plan_de_pruebas.md
└── src/
└── main.py
## Descripción de archivos clave

| Archivo | Descripción |
|---------|-------------|
| `README.md` | Presentación general del proyecto |
| `docs/propuesta.md` | Problema, justificación y arquitectura |
| `docs/caso_de_uso.md` | Flujos de uso del sistema |
| `docs/estructura_repositorio.md` | Este archivo |
| `docs/plan_de_pruebas.md` | Casos y estrategia de pruebas |
| `src/main.py` | Código principal del proyecto |

## Convenciones de nombres
- Archivos en minúsculas con guión bajo
- Documentación en español mexicano
- Código comentado en español

## Versionado
- Rama principal: `main`
- Commits descriptivos en español
- Un commit por funcionalidad o documento
